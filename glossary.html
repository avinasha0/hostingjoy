📖 Hosting Glossary · 2026

Web Hosting Glossary:
Every Term, Explained Simply

Hosting is full of jargon. This glossary covers every term you'll encounter when choosing or managing web hosting — explained in plain English, no technical background required.

A–C
Apache
The oldest and most common web server software. Slower than LiteSpeed and Nginx for most workloads. Still used by many budget hosts including Bluehost and GoDaddy.
Bandwidth
The amount of data transferred between your website and visitors. "Unlimited bandwidth" in hosting plans refers to data transfer, not connection speed. Rarely an actual constraint on modern plans.
CDN (Content Delivery Network)
A global network of servers that stores copies of your static files (images, CSS, JS) near your visitors. When someone in Tokyo visits your US-hosted site, the CDN serves files from a Tokyo server — reducing load time dramatically.
cPanel
The most popular hosting control panel. Lets you manage files, databases, email accounts, domains, and settings through a web interface. Most shared hosts use cPanel; Hostinger uses their own hPanel which many users prefer.
CLS (Cumulative Layout Shift)
A Core Web Vitals metric measuring how much page elements unexpectedly move during loading. A score below 0.1 is "Good." Caused by images without defined dimensions or content loading above existing content.
D–L
Dedicated Hosting
An entire physical server dedicated solely to your website(s). Maximum performance and control, but typically $80–500/month. Required only for very high-traffic sites (1M+ monthly visitors).
DNS (Domain Name System)
The internet's phonebook. Translates domain names (hostingjoy.com) into IP addresses (123.456.789.0) that servers use to communicate. DNS changes take 1–48 hours to propagate globally.
INP (Interaction to Next Paint)
A Core Web Vitals metric measuring how quickly a page responds to user interactions (clicks, taps). Under 200ms is "Good." Replaced First Input Delay (FID) as the interactivity metric in 2024.
LCP (Largest Contentful Paint)
A Core Web Vitals metric measuring how fast your main content appears. Under 2.5 seconds is "Good." The single most impactful metric for perceived page speed. Improved by optimizing hero images and reducing server response time.
LiteSpeed
A high-performance web server software (alternative to Apache/Nginx). Significantly faster for WordPress due to built-in server-level caching (LSCache). Used by Hostinger and HostArmada — one reason they consistently top our speed benchmarks.
M–S
Managed Hosting
Hosting where the provider handles server administration — updates, security, monitoring, and optimization. Typically costs more than unmanaged, but saves significant time. WP Engine and Kinsta are examples of managed WordPress hosts.
Nameservers
Special DNS servers that tell the internet where your domain's files are hosted. When you buy hosting, you update your domain's nameservers to point to your host (e.g., ns1.hostinger.com). This is how your domain finds your hosting server.
PHP
The programming language WordPress is built on. The version you run affects performance significantly. PHP 8.3 is up to 48% faster than PHP 7.4. Always run the latest stable PHP version your plugins support.
Shared Hosting
Multiple websites sharing one physical server's resources. Most affordable type of hosting ($2–10/month). Best for new websites and small businesses under 30,000 monthly visitors. The "bad neighbor effect" can occasionally impact performance.
SSL Certificate
Encrypts data between your server and visitors' browsers, enabling HTTPS. Required for any site handling user data, passwords, or payments. Google gives a slight ranking boost to HTTPS sites. All reputable hosts include free SSL via Let's Encrypt.
T–Z
TTFB (Time to First Byte)
How long it takes your server to respond to a browser request before sending any content. The foundational speed metric for hosting. Under 200ms is excellent. Above 600ms is problematic for SEO and user experience. This is the metric most influenced by hosting quality.
Uptime
The percentage of time your website is accessible online. Industry standard is 99.9% (about 8.7 hours/year of downtime). We recommend 99.95%+ (under 4.4 hours/year). 99.99% means under 53 minutes of downtime per year.
VPS (Virtual Private Server)
Dedicated CPU, RAM, and storage within a shared physical server. More expensive than shared hosting ($6–80/month) but provides guaranteed resources not affected by other users. Best when shared hosting can't handle your traffic.
WAF (Web Application Firewall)
Security software that filters malicious traffic before it reaches your site. Blocks SQL injection, XSS attacks, and brute force login attempts. Cloudflare provides a free basic WAF; Wordfence and Sucuri offer WordPress-specific WAFs.
WebP
A modern image format developed by Google. 25–35% smaller than JPEG and 50–80% smaller than PNG at equivalent quality. All modern browsers support WebP. Converting your images to WebP is one of the most impactful speed improvements you can make.
